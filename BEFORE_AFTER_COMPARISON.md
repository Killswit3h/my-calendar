### saveDraft - Event Validation

**BEFORE**:
```typescript

  setEvents([...events, {
    start: draft.start,
    end: draft.end,  // No validation
  }]);
```

**AFTER**:
```typescript
const saveDraft = () => {
  let finalEnd = draft.end;
  if (draft.allDay && draft.end) {

    const endDate = new Date(draft.end);
    // Handle edge cases
    if (endDate <= startDate) {
```
